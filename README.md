# Resume Analyzer

Resume Analyzer is a web application that helps job seekers evaluate their resumes. It extracts skills from uploaded resumes (PDF, DOCX, or images) 
and compares them with the skills required in a job description. The tool highlights matched skills, missing skills, extra skills, and provides 
a percentage score to indicate how well the resume matches the job.

## Features
- Upload resumes in PDF, DOCX, JPG, or PNG formats
- Automatic skill extraction from resumes
- Compare extracted skills with job descriptions
- Display matched, missing, and extra skills
- Calculate a resume match score
- Provides improvement suggestions for better job fit
- Easy-to-use, fast, and interactive UI

## Technologies Used
- HTML, CSS, JavaScript
- PDF.js for PDF processing
- Mammoth.js for DOCX parsing
- Tesseract.js for OCR (image resumes)
- Simple front-end UI for skill display and analysis

## How to Use
1. Upload your resume using the Choose Filebutton
2. Paste the Job Description in the textarea
3. Click Analyze Job Match
4. View your extracted skills, matched skills, missing skills, and resume score
